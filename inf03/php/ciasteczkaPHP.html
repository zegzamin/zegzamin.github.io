Ciasteczka w PHP

Ciasteczka (ang. cookies) to małe pliki przechowywane na komputerze użytkownika, które mogą być używane do przechowywania danych użytkownika na różnych stronach lub sesjach.

Tworzenie ciasteczka

Aby utworzyć ciasteczko w PHP, używamy funkcji setcookie().


<?php
setcookie("username", "Jan Kowalski", time() + (86400 * 30), "/");
?>

Odczytywanie ciasteczka

Aby odczytać ciasteczko, używamy superglobalnej tablicy $_COOKIE.


<?php
if(isset($_COOKIE["username"])) {
    echo "Witaj, " . $_COOKIE["username"];
} else {
    echo "Ciasteczko nie jest ustawione!";
}
?>
