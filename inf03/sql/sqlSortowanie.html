Sortowanie wyników

W arkuszach bardzo często pojawia się konieczność posortowania wyników, które otrzymamy z zapytania SELECT.
Służyć będzie do tego funkcja ORDER BY. Tą funkcję będziemy stawiać na samym końcu zapytania.
Możemy sortować rosnąco (od najmniejszego w górę) - dopisując słówko ASC lub malejąco (od największego w dół) dopisując słówko DESC.


    SELECT * FROM nazwa_tabeli ORDER BY nazwa_kolumny ASC;
    // Powyżej jest zastosowanie składniowo, sortowania rosnąco

    SELECT * FROM nazwa_tabeli ORDER BY nazwa_kolumny DESC;
    // Powyżej jest zastosowanie składniowo, sortowania malejąco

Zróbmy parę zadań na bazie z poprzedniej lekcji: 


    Nasze zadania:
        - Wyświetl czas, numer rejsu, kierunek, status z tabeli przyloty rosnąco
        - Wyświetl czas, numer rejsu, kierunek, status z tabeli przyloty malejąco

Pierwsze zadanie:

Zatem skoro będziemy wyświetlać to bierzemy SELECT: 


    SELECT nazwa_kolumny FROM nazwa_tabeli ORDER BY rodzaj_sortowania;

Teraz wprowadźmy nasze kolumny, czas, nr_rejsu, kierunek, status_lotu:


    SELECT czas, nr_rejsu, kierunek, status_lotu FROM nazwa_tabeli ORDER BY rodzaj_sortowania;

Wprowadźmy nazwę tabeli, czyli przyloty: 


    SELECT czas, nr_rejsu, kierunek, status_lotu FROM przyloty ORDER BY rodzaj_sortowania;

Teraz musimy dodać rodzaj sortowania, rosnąco, czyli ASC:


    SELECT czas, nr_rejsu, kierunek, status_lotu FROM przyloty ORDER BY czas ASC;

Super! Przejdźmy do drugiego zadania!


Drugie zadanie:

Mamy to samo polecenie co w pierwszym zadaniu, lecz zmieniamy rodzaj sortowania na malejący, czyli:


    SELECT czas, nr_rejsu, kierunek, status_lotu FROM przyloty ORDER BY czas ASC;

I teraz wystarczy zamienić ASC(rosnąco) na DESC(malejąco)


    SELECT czas, nr_rejsu, kierunek, status_lotu FROM przyloty ORDER BY czas DESC;

Zadania wykonane!
