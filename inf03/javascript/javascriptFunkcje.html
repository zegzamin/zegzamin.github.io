Funkcje w JavaScript

Funkcje to kod, który możemy używać wielokrotnie, bez pisania wszystkich jego linijek, a jedynie odwołując się do nazwy danej funkcji.
Z funkcji korzystaliśmy już w czasie tego kursu wielokrotnie - czy console.log, czy getElementByld - każda z nich jest funkcją.
Aby wywołać kod po naciśnięciu przycisku, również będziemy musieli utworzyć funkcję, do której go schowamy, by móc go wywołać.

Definiowanie funkcji

Funkcje definiuje się w JavaScript słowem kluczowym function, podając nazwę funkcji i otwierając nawiasy okrągłe ().

Przykład: Utwórz funkcję o nazwie zegzamin.


    function zegzamin() {
            //kod wykonywany po wywołaniu funkcji
    }

Wywoływanie funkcji z użyciem przycisku

Aby skorzystać z wywołania funkcji z użyciem przycisku, należy nadać przyciskowi atrybut onclick równy funkcji, np. onclick="zegzamin()" wywoła funkcję zegzamin
Zwróć uwagę, że w zapisie pojawiają się również nawiasy okrągłe - nie wystarczy podać samej nazwy funkcji, trzeba dodać dwa nawiasy.

    <button id="oblicz" onclick="oblicz()">OBLICZ</button>
