Modyfikowanie Atrybutów HTML Danego Elementu - JavaScript

Jednym z ciekawych motywów, który de facto nie pojawił się jeszcze na egzaminie, natomiast o który często pojawia się pytanie, jest edycja atrybutu, np. src za pomocą Javascript.
Jest to bardzo proste zagadnienie, wymagające jedynie wiedzy z lekcji 2.

Teoretycznie o zmianie atrybutu

Jak pamiętasz z lekcji o zmianie stylu, musieliśmy pobrać z dokumentu element za pomocą document.getElementById("id"), a potem po kropce wstawić albo style,
aby edytować styl CSS, albo innerHTML, aby edytować tekst elementu. W przypadku dowolnego atrybutu będziemy działać tak samo.
Po document.getElementById("id") po kropce podamy nasz atrybut.

Przykładowo: Zmień zdjęcie elementu o id zdjecie na plik o nazwie “zad1.png".


    document.getElementById("zdjecie").src = "zad1.png";

Teraz zróbmy zadanie, poniżej macie szkielet HTML


    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Lekcja 4 - JavaScript</title>
    </head>
                    
    <body>
        <h1> Testowanie </h1>
        <img src="" alt="" id="wybor">
        <button id="laptop">Laptop</button>
        <button id="lodka">Łódka</button>
        <button id="metro">Metro</button>
    </body>
    <script>
        //tu będzie nasz skrypt  
    </script>
    </html>

Pobierz sobie również 3 obrazki, będą one się znajdować w folderze z kodem do tej strony. Będą się one nazywać kolejno: laptop.jpg, lodka.jpg, metro.jpg

Teraz do naszych przycisków dodajmy sobie atrybuty onclick, każdy z tych przycisków będzie mieć inną funkcję


    <button id="laptop" onclick="laptop()">Laptop</button>
    <button id="lodka" onclick="lodka()">Łódka</button>
    <button id="metro" onclick="metro()">Metro</button>

    <script>
        //tu będzie nasz skrypt  
    </script>

Teraz napiszmy funkcję dla laptopa, która będzie brać src z img i będzie dawać obrazek laptop.jpg


    function laptop(){
        document.getElementById("wybor").src = "laptop.jpg";
    }

Teraz jeżeli naciśniemy przycisk Laptop, powinien nam się wyświetlić obraz "laptop.jpg"

Powtórzmy ten proces dla reszty przycisków: 


    function laptop(){
        document.getElementById("wybor").src = "laptop.jpg";
    }

    function lodka(){
        document.getElementById("wybor").src = "lodka.jpg";
    }

    function metro(){
        document.getElementById("wybor").src = "metro.jpg";
    }

W ten sposób możemy na stronie poprzez przyciski zmieniać style img, przez co zmieniają się nam obrazki
