JavaScript - Pobieranie Elementów

Każdy element w HTML znajduje się po renderze (wyświetleniu) strony w drzewiem DOM. To takie drzewo posiadające każdy element HTML, który został zdefiniowany.

Aby pobrać dany element ze strony, należy skorzystać z funkcji document.getElementById(), która pobiera element o danym indentyfikatorze.
Jeżeli element go nie posiada, to musimy mu go nadać - nazwa identyfikatora jest dowolna.

Identyfikator możemy nadać w następujący sposób: 


<p id="samochod"></p>

W ten sposób utworzyliśmy paragraf z identyfikatorem o nazwie "samochod"

Zmiana tekstu pobranego elementu.

Na pobranym elemencie możemy teraz operować, na przykład edytując jego tekst.
Po funkcji getElementById() stawiamy kropkę i odwołujemy się do atrybutu innerHTML, który odpowiada za tekst elementu.

Przykład: Edytuj paragraf o id samochod, tak, żeby tekst brzmiał "Multipla I"


    document.getElementById('samochod').innerHTML = "Multipla I"

Zatem w miejscu naszego paragrafu o identyfikatorze "samochod" powstanie napis: 


    Multipla I

Zmiana stylu pobranego elementu.

Możemy również edytować styl danego elementu - wystarczy, że odwołamy się do .style.nazwaWlasiwosci, gdzie nazwa właściwości to właściwość CSS, którą chcemy edytować.
Przypuśćmy, że chcemy zedytować kolor czcionki paragrafu na czerwony.

Przykład: Edytuj styl paragrafu o id samochod, tak, żeby tekst miał kolor czerwony.


    document.getElementById("samochod").style.color = "red";

Zatem w miejscu naszego paragrafu o identyfikatorze "samochod" styl zostanie zmodyfikowany na napis o kolorze czerwonym: 


    Multipla I

Zmiana dowolnego atrybutu pobranego elementu

Możemy edytować dowolny atrybut naszego elementu - wystarczy, że zamiast innerHTML czy style wpiszemy nazwę atrybutu, np. src bądź alt.

Przykład: Edytuj obrazek samochod, tak, żeby był wyświetlany plik porsche.jpg.


    document.getElementById("samochod").src = "porsche.jpg"; //Zakładając, że mamy ten obrazek

Pobieranie wartości pola edycyjnego

Najczęstszym motywem na egzaminie będzie pobranie wartości z danego pola edycyjnego i wykonanie na niej pewnych obliczeń, np. według cennika.
Aby to zrobi musimy nadać jakiś identyfikator polu edycyjnemu, do którego chcemy się odwoła pobrać go z użyciem document.getElementByld. Musimy wyciągnąć atrybut value.
Zapiszemy więc go do zmiennej. Całość będzie wyglądać następująco:


    let wartosc = document.getElementById(identyfikator).value;

Teraz mamy takie zadania: 


    Skrypt pobiera wartości z obu pól edycyjnych numerycznych
    Następnie oblicza koszt paliwa ze względu na rodzaj i liczbę litrów uwzględniając:
        Gdy rodzaj paliwa jest równy 1 - koszt jednego litra paliwa wynosi 4 zł
        Gdy rodzaj paliwa jest równy 2 - koszt jednego litra paliwa wynosi 3,5 zł
        W każdym innym przypadku koszt paliwa wynosi 0 zł
    Następnie skrypt wyświetla wynik działania pod przyciskiem OBLICZ według wzoru:
 „koszt paliwa: <wartość> zł, gdzie <wartość> oznacza obliczony wcześniej koszt paliwa

Zatem zacznijmy, mamy pobrać pola z dwóch pół edycyjnych, czyli z inputów o id ilosc oraz rodzaj.


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;

Tylko żeby operować na wartościach w naszym skrypcie, to najpierw musimy dodać jeszcze atrybut value do naszych inputów,
które będą naszymi początkującymi wartościami do tych elementów, więc:.


    <input type="number" id="rodzaj" value="1">
    <input type="number" id="ilosc" value="100">

Skoro też będziemy musieli policzyć koszt, to będziemy musieli dodać zmienną, która będzie przechowywać koszty


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;
    let koszt = 0;

Teraz musimy dodać warunki, przypomnijmy, dla paliwa typu 1, czyli input o id rodzaj będzie mieć value 1, to koszt paliwa będzie wynosić 4 zł, zatem:


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;
    let koszt = 0;

    if(rodzaj == 1) {
        koszt = ilosc * 4;
    }

Teraz dla paliwa o rodzaju 2, koszt będzie wynosić 3.5, zatem:


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;
    let koszt = 0;
    
    if(rodzaj == 1) {
        koszt = ilosc * 4;
    }
    if(rodzaj == 2) {
        koszt = ilosc * 3.5;
    }

Teraz ostatnia część warunku, jeżeli rodzaj nie będzie nosić ani 1 ani 2, to koszt wyniesie 0, zatem:


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;
    let koszt = 0;
    
    if(rodzaj == 1) {
        koszt = ilosc * 4;
    }
    if(rodzaj == 2) {
        koszt = ilosc * 3.5;
    }
    if(rodzaj != 1 && rodzaj != 2) {
        koszt = 0;
    }

Oczywiście możemy to wykonać za pomocą SWITCHa bądź ELSE IF oraz ELSE, lecz nie będziemy sobie komplikować.
Chcemy zaoszczędzić na egzaminie jak najwięcej czasu, więc zostanie nam więcej czasu na trudniejsze rzeczy jak np: kwerendy z baz itp.
Oczywiście za coś takiego nam nie odejmą punkty (chyba że będzie napisane jakiego typu warunków użyć, co praktycznie nigdy się nie zdarzyło)

Ostatnią częścią będzie wyświetlenie pod przyciskiem OBLICZ wyniku naszego skryptu, zatem: 


    let rodzaj = document.getElementById("rodzaj").value;
    let ilosc = document.getElementById("ilosc").value;
    let koszt = 0;
    
    if(rodzaj == 1) {
        koszt = ilosc * 4;
    }
    if(rodzaj == 2) {
        koszt = ilosc * 3.5;
    }
    if(rodzaj != 1 && rodzaj != 2) {
        koszt = 0;
    }

    document.getElementById("wynik").innerHTML = "koszt paliwa: " + koszt + " zł"; 
    //Ten '+' oznacza konkatenację, czyli łączenie łańcuchów znaków itp.

W wyniku tego działania, w paragrafie o identyfikatorze "wynik" pod przyciskiem OBLICZ będzie nam się pojawiać napis o wzorze: 


    koszt paliwa: {koszt} zł

W taki sposób poradziliśmy sobie z bardzo prostym skryptem do egzaminu "Stacja Paliw"
