Wprowadzenie do JavaScript. Zmienne, instrukcje warunkowe

EE.09/INF.03 składa się z 5 części, składających się na wszystkie umiejętności potrzebne na egzamin zawodowy. Tematem tej sekcji jest JavaScript.

JavaScript, podobnie jak pozostałe części, zaczynamy od podstaw, więc nie masz się czym martwić. Poruszymy wyłącznie kwestie potrzebne do egzaminu, pomijając zbędne zapychacze znajdujące się w podręcznikach. Zadbam jednak o nowoczesne podejście do tematu, więc wiedza, którą tu otrzymasz, będzie przydatna nawet po egzaminie w pracy z kodem.

JavaScript to język skryptowy - jest interpretowany po stronie przeglądarki (w porównaniu z PHP, który jest wykonywany po stronie Serwera).
Można odróżnić JavaScript od PHP na stronie, gdy na przykład klikamy jakiś element, to JavaScript wykona go bez przeładowania strony, a PHP zrobi to z przeładowaniem.


JavaScript i HTML - jak to połączyć?

Połączenie skryptu JavaScript z dokumentem HTML jest możliwe na dwa sposoby - poprzez atrybut src w sekcji script, więc osobny plik .js albo w sekcji <script>, bezpośrednio w dokumencie.


Zmienne w JavaScript

Zmienne w JavaScript, zupełnie jak w PHP, są automatycznie dopasowujące swój typ - nie ma potrzebny definiowania na przykład, że zmienna jest typu liczby całkowitej albo napisu
(oznacza to, że JavaScript jest językiem słabo typowanym co ma swoje minusy). JavaScript sam dopasowuje typ zmiennej do wartości, którą przyjmuje.

Aby utworzyć zmienną, musimy skorzystać ze słówka kluczowego let, a po nim podać nazwę zmiennej (nie korzystamy z var, gdyż jest przestarzałe i jest niezalecane).


    let zegzamin = "zdane";

console.log i proste debugowanie

Czasem zdarzają się sytuacje, że coś nie działa jak powinno. Możemy skorzystać z funkcji console.log, która zapisuje dany tekst bądź zmienną w konsoli deweloperskiej. Tam będziemy mogli podejrzeć, czy nasz kod wykonuje się poprawnie, czy zmienne przyjmują odpowiednią wartość i tak dalej


Instrukcje warunkowe w JavaScript

Na egzaminie pojawiło się wymagające użycia instrukcji warunkowych, warto więc o nich wspomnieć.
Konstrukcja jest identyczna jak w PHP i innych językach programowania: słówko if, a w nawiasie warunek.


    if (warunekPierwszy) {
        //kod wykonywany, jeżeli warunek jest prawdziwy
    } else if(warunekDrugi) {
        //kod wykonywany, jeżeli warunekPierwszy jest niepoprawny, lecz warunekDrugi jest prawdziwy
    }  else {
        //kod wykonywany, jeżeli żaden z powyższych warunków nie jest prawdziwy
    }

!! Uwaga !! Skrypt chcemy wyświetlać pod elementem body (gdyż nie chcemy go wyświetlać, on ma dokonywać obliczeń itp).
W head też nie dodajemy <script>, gdyż w tedy byłoby to interpretowane że coś wysyłamy przeglądarce, czego nie chcemy.

Zatem pomiędzy znaczniki <script> </script> wpisujemy sobie nasz kod.

Zróbmy sobie na przykład zmienną o nazwie zmienna, niech będzie mieć wartość 10, zatem:


    let zmienna = 10;

Teraz na przykład możemy sobie zmodyfikować naszą zmienną, dodajmy sobie na przykład do niej 10, zatem:


    let zmienna = 10;
    zmienna = zmienna + 10;

Teraz wyświetlmy za pomocą console.log() zmienną w konsoli:


    let zmienna = 10;
    zmienna = zmienna + 10;
    console.log(zmienna);

Teraz jeżeli na naszej stronie na której mamy ten skrypt podłączony, klikniemy "Zbadaj" oraz przejdziemy do zakładki "Konsola", wyświetli się nam wynik: 


    20

Warunki IF

Niech nasza zmienna będzie równa na przykład 10


    let zmienna = 10;

Teraz sobie dodajmy do tego warunek, na przykład, jeżeli nasza zmienna wynosi 10, to niech wypisze, że "Zmienna jest równa 10!"


    let zmienna = 10;
    if(zmienna == 10) { //Uwaga!! Porównując zmienną do liczb używamy == a nie =
        console.log("Zmienna jest równa 10!");
    }

Wynik tego skryptu w Konsoli przeglądarkowej wyświetli nam napis: 


    Zmienna jest równa 10!

Oczywiście możemy to sobie rozbudować, możemy na przykład do tego IFa dodać warunek ELSE IF oraz ELSE (ELSE będący zawsze na końcu)

Zmieńmy sobie zatem naszą zmienną 'zmienna', żeby przechowywała wartość 9, oraz dodajmy IFa sprawdzającego, czy nasza 'zmienna' ma wartość 10.


    let zmienna = 9;
    if(zmienna == 10) {
            console.log("Zmienna jest równa 10!");
    } else {
        console.log("Zmienna nie jest równa 10.")
    }

ELSE zawsze będzie wykonywać jakiś kod, jeżeli powyższe warunki były nieprawdziwe, zatem wynikiem powyższego kodu będzie napis w Konsoli:


    Zmienna nie jest równa 10.

Oczywiście nie jest równa, gdyż dane przypisane zmiennej 'zmienna' nie wynosi 10, lecz 9.
