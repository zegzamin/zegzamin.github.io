Alerty, Prompty i Parsowanie w JavaScript

Alert, to takie okienko które się pokazuje na górze strony. Często jest kojarzony ze scamerskimi stronami. Ogólnie sam w sobie alert jest przestarzały i rzadko się go używa, lecz CKE uważa że tak jak PHP, są bardzo potrzebne programistom (co w rzeczywistości wiemy jak wygląda)

Aby skorzystać z funkcji alertu, piszemy w skrypcie: 


    function wyswietlAlert() {
            alert("Przykładowy napis");
    }

Oczywiście bez wywołania funkcji w której jest alert, to nic się nie pokaże na stronie. Możemy sobie na przykład wywołać to za pomocą jakiegoś przycisku:

Zróbmy zatem dwa przyciski, jeden będzie do naszego alerta, a drugi do prompta, którego zaraz omówimy: 


    <button onclick="wyswietlFunkcje()">Pokaż alert</button>
    <button onclick="wyswietlPrompt()">Pokaż prompt</button>

Po naciśnięciu przycisku "Pokaż Alert", wyświetli się nam alert z tekstem, który wpisaliśmy w cudzysłów w jego nawiasy.

Prompt

Prompt, działa podobnie jak alert, lecz on przyjmuje dane. Wyświetla się w bardzo podobnym okienku i przyjmuje dane tekstowe (nie liczbowe)

Napiszmy zatem prompta w naszym skrypcie: 


    function wyswietlAlert() {
        alert("Przykładowy napis");
    }

    function wyswietlPrompt() {
        let wynik = prompt("Wpisz jakiś napis: ");
    }

Musimy jeszcze utworzyć zmienną, w której będzie przechowywany wynik prompta, dlatego też stworzyliśmy sobie zmienną "wynik".

Teraz jeżeli naciśniemy przycisk "Pokaz Prompta", wyświetli się nam okienko, w które możemy wpisać jakieś dane tekstowe.

Jeżeli jednak chcemy dokonać jakiś obliczeń na naszym wyniku, musimy wynik Parsować.

Co to oznacza? W skrócie jest to zmiana jednego typu danych na drugi, w tym przypadku będziemy zmieniać nasz wynik czyli tekst na liczbę, czyli na przykład int, zatem:


    function wyswietlAlert() {
        alert("Przykładowy napis");
    }

    function wyswietlPrompt() {
        let wynik = prompt("Wpisz jakiś napis: ");
        document.getElementById("wynik").innerHTML = parseInt(wynik); 
        //zakładając, że mamy jakieś pole z identyfikatorem "wynik" na stronie
    }

W ten sposób wyświetlamy nasz wynik prompta jako liczba, przez co możemy dokonywać wielu różnych operacji na naszej zmiennej wynik, na przykład: 


    function wyswietlAlert() {
        alert("Przykładowy napis");
    }

    function wyswietlPrompt() {
        let wynik = prompt("Wpisz jakiś napis: ");
        document.getElementById("wynik").innerHTML = parseInt(wynik) + 1; 
    }

Na przykład powyżej zmodyfikowany kod, po dodaniu 1 za parsowaniem doda nam 1 do liczby wpisanej przez użytkownika
